# NPM-Random-Messages

[This is just a practice project] Get inspire yourself by random messages 'Thanks for code, you are building the future', 'Stop talking and lets code', etc.

## Install

```npm
npm i -g platzi-npm-random-messages
```

# Usage

```bash
message
```
